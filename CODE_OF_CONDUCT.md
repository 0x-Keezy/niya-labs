# Code of Conduct

Niya Labs adopts the **[Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)** as its code of conduct.

The full text (standard across thousands of open-source projects) lives at the link above. Please read it before contributing.

## Scope

The Covenant applies anywhere the Niya Labs community interacts:

- This GitHub repository (issues, pull requests, discussions, code review comments)
- The `/admin` panel logs and any maintainer-facing tools
- Social channels where Niya Labs is officially represented (currently [@NiyaAgent](https://x.com/NiyaAgent))
- Any event, stream, or call organised under the Niya Labs name

## Reporting

If you experience or witness behaviour that violates the Covenant, report it privately:

- **GitHub Security Advisory** (preferred for anything sensitive): [new advisory](https://github.com/0x-Keezy/niya-labs/security/advisories/new)
- **DM on X**: [@NiyaAgent](https://x.com/NiyaAgent)

All reports are reviewed by the maintainers promptly and kept confidential. We will acknowledge within 48 hours.

## Enforcement

Maintainers follow the Covenant's four-step enforcement ladder (Correction → Warning → Temporary Ban → Permanent Ban) and document the specific action taken. Good-faith contributors who make a mistake and correct course stay welcome.
